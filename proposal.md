## Description
This program will compile a list of videos or images with into one video. It will also add an intro panel with a logo, artist's information, and title.

## Features
- Feature 1
	single interface that allows input and images to be inported into the program
- Feature 2
	compiles the videos together with or without watermark
- Feature N 
	saves to selected directory of choosing
